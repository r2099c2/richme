# Richme 工具链与决策

- **已定**：前端包管理使用 **pnpm**；Python 环境与依赖使用 **uv**（`uv sync` / `uv run`）。维护者本机通常 **已安装** pnpm 与 uv，实装步骤中不必默认再指导安装，仅需校验版本或指向官方文档即可。
- **确认原则**：当出现与上述同类的 **可选方案**（例如另一种包管理器、另一种 Python 工具、ORM/迁移工具的等价替代、部署形态的多种选择等）时，**不要自行替维护者拍板**；先用简短选项 **向用户确认** 后再写入计划或改代码。
