# Richme Web

Vite + React + TypeScript + Tailwind CSS v4. Package manager: **pnpm**.

## Prerequisites

- Node.js 20+
- [pnpm](https://pnpm.io/installation) 9.x (`package.json` 中 `packageManager` 与 Corepack 对齐)

## Commands

```bash
pnpm install
pnpm dev
```

- Build: `pnpm build`
- Preview: `pnpm preview`

复制 `.env.example` 为 `.env` 可配置 `VITE_API_BASE_URL`（步骤 4 使用）。
